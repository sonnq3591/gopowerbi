Sales Performance Dashboard

Power BI | Sales Analytics | Dashboard Engineering

Overview

This project involved designing and deploying an interactive Power BI dashboard for a global retail client. The goal was to provide clear visibility into sales KPIs across multiple regions and product categories, empowering the sales team with actionable insights.

Key Features

  • Dynamic filtering and drill-downs
  • Advanced DAX measures for YoY & MoM comparisons
  • Custom visualizations for product performance
  • Data refresh automation via Power BI Gateway

Technology Stack

Power BI DAX SQL Server ETL Data Modeling
