Overview

These Power Automate flows enable a range of intelligent business automation scenarios, from document processing and form handling to integrating AI and real-time notifications via Microsoft Teams. They demonstrate how low-code workflows can efficiently streamline operations using Microsoft 365, SharePoint, OpenAI, and more.

Key Automations

  • Manually trigger document creation workflows with Microsoft Word and send results via email or store to SharePoint.
  • Automate survey/form response collection, file creation, and Power BI dataset refreshes.
  • Use switch logic to tailor workflows per business region or input type with dynamic content merging.
  • Recognize text in scanned PDFs, generate prompts, send to OpenAI for summarization, and notify users in Teams.
