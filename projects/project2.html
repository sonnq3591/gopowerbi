AI-Enhanced Analytics Platform

End-to-end Azure data platform ingesting from ERP systems, APIs, and internal sources. Integrated OpenAI for automated employee performance analysis. Final data served to Synapse + Power BI.

Overview

This platform integrates Azure data ingestion, OpenAI-based analysis, and high-performance reporting through Synapse and Power BI. The pipeline handles sensitive employee comments by securing access with Azure Key Vault, using Azure Functions to call OpenAI for performance scoring, and persisting results into Synapse for executive reporting.

Key Features

  • Secrets retrieved using Azure Key Vault for secure configuration
  • OpenAI integration through Azure Function for scoring comments
  • Cleaned results stored in ADLS, enriched via ADF
  • Data loaded into Synapse Dedicated SQL Pool for fast analytics
  • Visualizations delivered via Power BI dashboards
